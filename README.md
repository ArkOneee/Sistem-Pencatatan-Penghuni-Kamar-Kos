# Sistem-Pencatatan-Penghuni-Kamar-Kos
Program ini bertujuan untuk mempermudah pengelolaan data suatu kamar kos dimana nantinya program akan mencatat data penghuni, dan melacak status pembayaran bulanan. Program ini juga mempermudah pemilik kos untuk mencari data (kamar/penghuni) dan membuat laporan rekapitulasi yang diurutkan berdasarkan nomor kamar.
